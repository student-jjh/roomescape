# 사건 17호: 추리 방탈출 (Vanilla JS)

브라우저 기반 추리형 방탈출 게임입니다.  
바닐라 JavaScript로 구현했으며, 챕터를 순차적으로 진행하면서 단서를 해석해 최종 잠금 코드를 풀어야 합니다.

## 프로젝트 소개
`사건 17호`는 8년 전 미제로 종결된 사건을 재수사하는 설정의 추리 게임입니다.  
플레이어는 제한 시간 안에 단서를 복원하고, 공모자의 조작 기록을 추적해 출구 잠금을 해제해야 합니다.

## 주요 기능
- 챕터형 진행: 무전 로그 -> 퍼즐 -> 인터루드 스토리
- 단서 이미지 기반 추리 퍼즐
- 타이머(15분), 힌트 시스템, 저장/이어하기
- 퍼즐별 오답 통계(막힌 구간 확인)
- 사운드 효과 및 화면 전환 연출

## 기술 스택
- HTML
- CSS
- Vanilla JavaScript (모듈 번들러 없이 구성)

## 프로젝트 구조

## 로컬 실행
아래 중 하나로 실행할 수 있습니다.

1. `index.html`을 브라우저에서 직접 열기
2. 로컬 서버 실행 후 접속
   ```bash
   python3 -m http.server
   ```
